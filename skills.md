# Skills.md — Tech Stack & Development Skills

## Frontend Skills
- **HTML5** — Semantic markup, accessibility best practices
- **CSS3** — Flexbox, Grid, CSS variables, animations, responsive design
- **JavaScript (ES6+)** — Arrow functions, destructuring, async/await, modules
- **React.js** — Functional components, hooks (useState, useEffect, useContext, useCallback), React Router v6
- **Axios** — HTTP requests, interceptors, error handling
- **Chart.js + react-chartjs-2** — Line charts, bar charts, dashboard data visualization
- **Poppins Font** — Google Fonts integration for professional typography

## Backend Skills
- **Node.js** — Server-side JavaScript runtime
- **Express.js** — REST API, middleware, routing, error handling
- **JWT (jsonwebtoken)** — Stateless authentication, token generation and verification
- **bcryptjs** — Password hashing and comparison
- **Sequelize ORM** — Model definition, associations, migrations, querying with Op
- **mysql2** — MySQL driver for Node.js
- **dotenv** — Environment variable management
- **cors** — Cross-Origin Resource Sharing configuration
- **nodemon** — Development auto-restart

## Database Skills
- **MySQL** — Relational database design, normalization
- **SQL** — DDL (CREATE TABLE), DML (INSERT, SELECT, UPDATE, DELETE), foreign keys, indexes
- **Sequelize Models** — DataTypes, associations (hasMany, belongsTo), sync

- **JWT Auth Flow** — Login → token → protected routes

- **Protected Routes** — React PrivateRoute pattern
- **Auth Middleware** — Express middleware for token validation
- **CORS** — Restricting allowed origins
